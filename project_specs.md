# Portfolio Website Master Specification

## 1. Project Goal
Build a highly aesthetic, responsive, and modern personal portfolio website. The visual language should be bold, utilizing heavy typography, high contrast, and clean grid alignments similar to top-tier graphic design portfolios, but tailored for a software developer and robotics engineer.

## 2. Tech Stack
* Frontend: HTML5, CSS3, JavaScript (ES6+)
* Styling: CSS Grid/Flexbox, Vanilla CSS (or Tailwind CSS if preferred for speed)
* Animations: CSS Keyframes (for marquees) and Intersection Observer API (for scroll-fade effects).
* Deployment target: GitHub Pages.

## 3. Design System & Layout
* **Color Palette:** High-contrast monochrome (deep blacks, crisp whites, subtle grays) with one distinct accent color.
* **Typography:** Use a bold sans-serif font (like Inter or Space Grotesk) for massive headers, and a clean secondary font for body text.
* **UI Elements:** Use minimalist borders, sharp corners, and hover state scaling for project cards. 

## 4. Content Architecture & Sections
The single-page application must include the following sections in order:

* **Hero Section:** Massive typography stating "Arka Datta". Subtitle: "Full-Stack Web Developer & Robotics Engineer". Include a clean "Get in touch" call-to-action.
* **Scrolling Marquee:** An infinite horizontal scrolling banner listing core technologies: MongoDB • Express.js • Node.js • React • Python • Java • C • C++
* **Featured Projects (Grid Layout):**
  * Project 1: AI Chatbot for Movie Reviews (Developed during IIT Kanpur AI/ML Internship).
  * Project 2: Automated Toll Gate Development.
  * Project 3: Miniature 6 DOF Robotic Arm.
  * Project 4: Custom Car Brand Website.
* **Experience & Education (Two-Column Layout):**
  * **Education:** 
    * College: University School of Automation and Robotics (B.Tech in Automation and Robotics, currently in 3rd Year).
    * Schooling: Kalyani Public School (12th Grade).
  * **Achievements:**
    * Sarcastic achievements (e.g., "Successfully Failed NEET" to pivot to robotics, high coffee-to-code conversion rate).
  * **Experience & Certifications:** 
    * AI/ML Intern at IIT Kanpur.
    * Completed CS50x from Harvard University.
* **My Creative Stack (Unified Logo Grid):** 
  * An interactive, highly responsive grid of technology logos with smooth hover scaling effects, featuring:
    * HTML5, CSS3, JavaScript, Node.js, Express.js, MongoDB, Java, Python, C, C++, and Git.
* **Footer:** Minimalist contact links (GitHub, LinkedIn, Email).
